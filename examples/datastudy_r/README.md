This folder contains the set up for a reproducible workflow as described by https://github.com/FellowsFreiesWissen/computational_notebooks.git

The file structure is organized as such:

``` text
project_name
  ├── README.md  
  ├── main.Rmd  
  ├── results
  |     ├── README.md
  |     ├── data
  |     |     ├── raw
  |     |     └── processed
  |     └── scripts
  ├── text
  |     ├── README.md
  |     ├── main.doc
  |     ├── figures
  |     ├── tables
  |     └── supplementary
  └── submission
        ├── README.md
        ├── journal1
        |     └── first
        └── journal2
              ├── first
              └── revisions
```
