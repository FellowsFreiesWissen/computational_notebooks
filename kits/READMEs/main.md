This folder contains the set up for a reproducible workflow as described by <https://github.com/ludmillafigueiredo/computational_notebooks>.

The file structure is organized as such:

```
projet_name
  |-- README.md
  |-- main.Rmd
  |-- results
  |   |-- README.md
  |   |-- data
  |   |   |-- raw
  |   |   |-- processed
  |   |-- scripts
  |-- text
  |   |-- README.md
  |   |-- main.doc
  |   |-- figures
  |   |-- tables
  |   |-- supplementary
  |-- submission
  |   |-- README.md
  |   |-- journal1
  |       |-- first
  |   |-- journal2
  |       |-- first
  |       |-- revisions
```
